# Примеры работы с СУБД

Здесь приведены основные сценарии использования системы.

## Интерактивный режим

Запустите основное приложение без аргументов:
```bash
./build/course_work
```
Пример сессии:
```sql
CREATE DATABASE school;
USE school;
CREATE TABLE students (id INT INDEXED, name STRING, age INT DEFAULT 18);
INSERT INTO students (id, name, age) VALUE (1, "Ivan", 20);
SELECT * FROM students WHERE age > 18;
EXIT;
```

## Пакетный режим (выполнение скриптов)

Вы можете передать SQL-файл в качестве аргумента:
```bash
./build/course_work scripts/demo.sql
```

## Работа через сервер

1. Запустите сервер:
   ```bash
   ./build/course_server 8080
   ```

2. Используйте клиент для выполнения скрипта:
   ```bash
   ./build/course_client 127.0.0.1 8080 example/test1.sql admin admin
   ```

3. Или используйте `curl` для отправки запросов напрямую:
   ```bash
   # Логин
   TOKEN=$(curl -s -X POST http://127.0.0.1:8080/login      -H 'Content-Type: application/json'      --data-binary '{"user":"admin","password":"admin"}' | jq -r .result)

   # Запрос
   curl -X POST http://127.0.0.1:8080/query      -H "Authorization: Bearer $TOKEN"      --data-binary 'USE demo; SELECT * FROM users;'
   ```
